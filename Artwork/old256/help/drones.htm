Drone Documentation

 

All drones

STOP

Syntax: [dronename] stop

Description: The drone stops whatever it is doing and returns to the default behavior. For turrets, this means firing at enemies. For most drones, they will just sit still.

STATUS

Syntax: [dronename] status

Description: The drone will respond via chat describing the action it is currently engaged in.

 

DESIGNATE BASE

Syntax: [dronename] designate base [object]!

Description: The drone will choose the designated object (base) as its point of return for repairs, trade, and amublence service.

 

 

Turret drone

GOTO

Syntax: [dronename] goto [object]!

 

FOLLOW

Syntax: [dronename] follow [object]!

Description: The turret drone will find the object, figure out a basic path to the object (avoiding obstacles) and try to match it’s speed and velocity. The only difference is that the goto command will be considered completed when the drone reaches its target, and the follow will stay active.

 

KILL

Syntax: [dronename] kill [object]!

Description: The drone will go and attack an object with its weapon until it is out of ammo, its shields are at less than 0%, or its hull integrity is less than 50%.  The drone will follow the object wherever it goes.

 

DEFEND

Syntax: [dronename] defend [object]!

Description: Defend this object using its primary weapon(s).  Never let the object get out of its scan range, but pursue attackers to that point.  Die defending if necessary.

MINE

Syntax: [dronename] mine [object]!

Description: The mining drone will decide whether the target is suitable to mine. If it is (a planet or an asteroid), then it will go to it, mine it (for the length specified by limit), and then find the closest friendly starbase that it can use to unload.

If the target is a spacestation, then it will try to unload its resources at the starbase.

If the target is not any of the above, then it will goto or follow the object as appropriate.

PATROL

Syntax: [dronename] patrol [object]!

Description: The drone will go from its current location to a specified arbitrary point, or other object, turn around and go back.  If an enemy comes into range, then the drone will attack, but not go through an Aleph, instead it will return to patrol.
